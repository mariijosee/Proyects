# Computer Engineering Projects 

This repository contains a curated selection of my most significant projects developed during my Computer Engineering degree, focusing on **Artificial Intelligence, Algorithm Optimization, and Computer Vision.**

## Project Categories

### [Metaheuristics & Optimization](./metaheuristics-MDP)
* **Maximum Diversity Problem (MDP):** Implementation of Greedy, Tabu Search, GRASP, and Genetic/Memetic algorithms to solve NP-Hard challenges.
* **Tech:** C++, Optimization Theory.

###[Advanced Algorithms](./advanced-algorithms)
* **Paradigm Exploration:** Efficient solutions using Divide & Conquer ($O(n \log n)$), Dynamic Programming, and Backtracking.
* **Analysis:** Empirical benchmarking and Big O complexity validation.
* **Tech:** C++.

### [Computer Vision](./computer-vision)
* **Pattern Recognition:** Biological particle classifier (SVM/K-NN) using HOG and PCA.
* **Augmented Reality:** Real-time 3D pose estimation (PnP algorithm).
* **Tech:** C++, OpenCV.

### [Data Mining](./data-mining)
* **Machine Learning Pipelines:** Dimensionality reduction and robust benchmarking using non-parametric statistical tests (Wilcoxon/Friedman).
* **Tech:** Python, Scikit-Learn, Pandas.
